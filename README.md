# Kahoot Bat Mitzva - Ver02

MVP מקומי לחידון בסגנון Kahoot, עם שליטה מלאה של המנחה בקצב.

הגרסה הזו בכוונה פשוטה: Flask בלבד, בלי בסיס נתונים ובלי WebSocket. הדפדפנים מתעדכנים מול השרת המקומי כל חצי שנייה, כדי להגיע מהר לבסיס עובד.

## התקנה והרצה

1. לפתוח Terminal בתוך תיקיית `Ver02`.
2. ליצור סביבה וירטואלית:

```powershell
python -m venv .venv
```

3. להפעיל אותה:

```powershell
.\.venv\Scripts\Activate.ps1
```

4. להתקין חבילות:

```powershell
pip install -r requirements.txt
```

5. להריץ:

```powershell
python app.py
```

מסך מנחה:

```text
http://localhost:5000/host
```

מסך משתתף מאותו מחשב:

```text
http://localhost:5000/player
```

## התחברות מטלפונים

הטלפונים והמחשב צריכים להיות על אותו Wi-Fi.

כדי למצוא את כתובת ה-IP המקומית ב-Windows:

```powershell
ipconfig
```

לחפש את השורה `IPv4 Address`, למשל `192.168.1.23`.

בטלפון פותחים בדפדפן:

```text
http://192.168.1.23:5000/player
```

השרת גם מדפיס במסך ההרצה כתובת מוצעת לטלפונים.

## ניהול שאלות

השאלות נמצאות בקובץ `DataQuestions.py`, בתוך `questions_list`.

אם לשאלה יש תמונה, לשים את הקובץ בתיקיית `images` ולכתוב את שם הקובץ בשדה `Image`.
